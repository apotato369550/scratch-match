What is Scratch Match?
A locally-bound, AI-assisted employment matchmaker for the Province of Cebu.
Upload a CV. Post a vacancy. Ask the advisor. Everything runs on a single machine inside the province — no cloud lock-in, no data leaving Cebu.

09
The stack

Four pieces. All run on one machine.

App
Next.js
App Router, server actions, one Node process — no microservices.
Records
SQLite
Single file at data/app.db. Cheap, file-based, zero ops.
Vectors
Qdrant
Two collections (CV chunks, job chunks), cosine similarity, 768 dims.
Brains
Ollama
Mistral 7B for chat, nomic-embed-text for embeddings. No API keys.
No cloud. No API keys. No data leaves the machine.

From "as-is" to "to-be"

Refinement strategy.

Not a redesign — a tightening. Same pitch, same shape, more of the feature paths actually working end to end.

  • Minimalist navy & gold visual language
